# ExamPrep Telegram Mini App

Мини-приложение для подготовки к экзаменам внутри Telegram.

## Ссылка на приложение:
https://necromasteryxd.github.io/examprep-tg-app/

## Как подключить к Telegram боту:
1. Откройте @BotFather в Telegram
2. Найдите вашего бота
3. Отправьте команду: `/setmenubutton`
4. Введите URL: https://necromasteryxd.github.io/examprep-tg-app/
5. Готово! В меню бота появится кнопка "Web App"

## Технологии:
- HTML5, CSS3, JavaScript
- Telegram Web App SDK
- GitHub Pages для хостинга
